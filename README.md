## Inspirasi
Permainan ini terinspirasi dari "Magic 8 Ball", yaitu mainan klasik berbentuk bola hitam yang digunakan banyak orang untuk bersenang-senang saat ingin mendapatkan jawaban atas pertanyaan mereka.
Biasanya dimainkan seperti ini:
Seseorang mengajukan pertanyaan, misalnya:

"Apakah aku akan menang undian hari ini?"

"Apakah dia suka padaku?"

"Haruskah aku makan mi atau nasi malam ini?"

Lalu, mereka mengguncang bola ajaib â€” atau dalam versi aplikasi ini, klik bola di layar.
Gambar bola akan berubah, yang seolah-olah memberikan jawaban acak, seperti:

"Ya!"

"Tidak!"

"Mungkin..."

"Tanya lagi nanti."

"Sudah pasti!"

Permainan ini sering dimainkan saat nongkrong bareng teman, untuk iseng-iseng seru, atau bahkan hanya untuk menghibur diri sendiri saat bingung membuat keputusan.

## Cara kerja aplikasi:
Tampilan Awal:
Aplikasi akan menampilkan gambar bola di tengah layar.
Judul di bagian atas bertuliskan "Tanya Daku Apa Saja".

Cara Menggunakan:
Klik gambar bola tersebut.
Setelah diklik, gambar bola akan berubah secara acak (ada 5 kemungkinan gambar berbeda).
Setiap gambar bisa diartikan sebagai jawaban acak, cocok untuk game tanya-jawab atau iseng-iseng seperti "bola ajaib".

Tujuan:
Memberikan jawaban acak secara visual.
Seru digunakan untuk menanyakan pertanyaan seperti "Apakah saya akan beruntung hari ini?" lalu klik bolanya untuk lihat "jawaban".
